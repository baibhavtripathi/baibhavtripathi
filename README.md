### Hey there! 👋

I'm an **Agentic Systems Engineer** who sits at the intersection of enterprise process automation and modern AI. I design and build intelligent workflow systems using **Camunda 8** for orchestration, **Java Spring Boot** for microservices, and **LLM-powered agents** for autonomous decision-making — turning complex business requirements into self-operating AI pipelines.

⚙️ **What I build:** Production-grade agentic workflows where Camunda BPMN handles process orchestration and AI agents handle the decisions that can't be hardcoded

🔁 **My edge:** Years of real Camunda + Java experience maps directly to what the AI world now calls agent orchestration, tool calling, and event-driven agent systems

🎯 **Open to:** Agentic systems, AI backend, and LLM integration roles
